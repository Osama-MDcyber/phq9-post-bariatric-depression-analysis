# PHQ-9 Post-Bariatric Depression Analysis

R code and report for a cross-sectional study of depression prevalence and risk 
factors among 186 post-bariatric surgery patients using PHQ-9 scoring.

## Contents

- Data cleaning and preprocessing including BMI calculation, surgery type 
  classification, and PHQ-9 factor coding.
- Baseline demographics table with descriptive statistics (median [IQR], frequencies).
- PHQ-9 severity distribution and depressive syndrome classification (MDD vs. Other).
- Univariate and multivariable binary logistic regression for predictors of depression
  (crude and adjusted odds ratios with 95% CIs).
- Publication-ready tables and figures.
